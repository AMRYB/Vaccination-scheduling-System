# Vaccination Scheduling System🧑‍⚕️

## Project Overview
The Vaccination Scheduling System is a pivotal tool in automating the vaccination process in Egypt. Developed to assist the Ministry of Health's special vaccination units, this system simplifies the registration and scheduling of vaccinations for both administrators and users, ensuring a more organized and effective vaccination campaign.

## How to Use
- **Registration:** Users need to register in the system by entering their personal details.
- **Scheduling Appointments:** The program schedules vaccination appointments based on vaccine availability and user priority.

## System Capabilities

### User Roles
- **Admin**
- **Users**

### Admin Capabilities
- **Login:** Securely access the system with their profile credentials (Id, name, email, password).
- **Manage Vaccination Centers:** Add or remove centers, each with its unique Id, Name, Address, and a list of available vaccines.
- **Search Functionality:** Look up vaccination centers by name.
- **User Management:** View a list of registered users and their reservation status.
- **Reservation Approval:** Accept users' reservation requests and assign dates for vaccination.

### User Capabilities
- **Registration:** Sign up by providing personal information (Id, name, email, password, phone number, national id).
- **Access Information:** View a list of vaccination centers after logging in.
- **Reserve Vaccination:** Make a reservation with details (user id, vaccination center id, vaccine name).
- **View Vaccination Date:** Check the assigned date for vaccination post-admin approval.

## Features
- Two distinct user roles with specific permissions.
- Admin control panel for comprehensive system management.
- User-friendly interface for easy appointment scheduling.
- Secure authentication and user data protection.
- Real-time updates and notifications on vaccination appointments.
